# Literature

Academic literature for the conditional floor-plan generation project, sourced from the
[Modified Swiss Dwellings (MSD)](https://archilyse.standfest.science/modified-swiss-dwellings) project page.

Openly-accessible papers are stored here as both the original `.pdf` and a converted `.md`
(text extracted with [`pymupdf4llm`](https://pypi.org/project/pymupdf4llm/)). Paywalled papers
are **linked only, not stored** — follow the DOI / publisher link to access them.
